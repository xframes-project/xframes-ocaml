# xframes-ocaml

## Instructions

### Install opam

#### Windows

I strongly recommend installing [opam](https://opam.ocaml.org/) through [scoop](https://scoop.sh/).

`scoop install opam`

### Install ocaml

- `opam init`
- `opam switch create 4.14.0`

### Install dune

`opam install dune`

### Running the application

`dune exec`
